A little bit about me

I got my first proper computer back in 1988; An Amstrad CPC 6128 with 128k memory and an external tape deck.

This blue screened wonder gave me my first introdution to comuter programming in the form of BASIC, and made me fall in love with computing.

Fast forward to 1995 and I built my first website with geocities complete with background MIDI music and flashing stuff all over the screen! Experimented with networking and discovered Linux and the delights of open source software.

Currently im working on my skillset in:

  • Python
  • Javascript
  • SQL
  • Git

I also volunteer at a local CodeClub and do my part to help bring computer science to kids using code and BBC Micro:bits.

This blog is my way to document the path I take to contribute to open source projects

My name is Leigh Wilson and I am DevOps.
